みやこサンセットビーチについて。


「みやこサンセットビーチ」は宮古島屈指の夕日スポットとして知られているサンセットビーチです。伊良部大橋を一望できるサンセットスポットとしても有名で、水平線に落ちる夕日とそれを包み込む海は非常に幻想的な景色です。

市街地から近い人口のビーチで、季節限定で海水浴場としても解放されており、海水浴を楽しむこともできます。ハブクラゲ侵入防止ネットが設置しており（7月～10月）安心してご利用いただけます。

水際で魚を見ることもでき、釣りスポットとしても人気があります。

また、カヤックやSUPなどのアクティビティも充実しており、大自然の中でウォータースポーツを満喫できます。

市街地や宮古空港からアクセスが便利で、観光客だけではなく地元の方も集まるビーチスポットであり、駐車場やトイレも完備しております。

ビーチ周辺には長い遊歩道や公園もあり、ウォーキングやレジャーを楽しむ方々が多く訪れます。また、お子様連れの方も安心してご利用いただけます。

家族や友人と訪れる夏のリゾートとして最適な「みやこサンセットビーチ」。幻想的で美しい大自然の景色を楽しんでみませんか？

    Sup Program

    1. ミルキーウェイSUP

    トゥリバーエリアと宮古島本島との間にある水路を探検します。本島側には、波浪の浸食によって作られた海岸の洞窟(海蝕洞)を見ながら、トゥリバー側には、ヒルトンの建物を見ながら、時代の今昔を体験できます。
    【参加費: 6,500円】
    【所要時間: 2時間30分】
    【開催場所: トゥリバー水路】
    【開催時間: 9時30分、13時30分 1日2回】

    2. Fun SUP 伊良部大橋をくぐってウミガメに会いに行こう

    サンセットビーチを出向して、左に曲がり無料で渡れる日本最長の橋、伊良部大橋をくぐって海岸線をクルージングします。海面から顔を出すウミガメ達に出会えることも。
    【参加費: 7,000円】
    【所要時間: 2時間30分】
    【開催場所: サンセットビーチ～伊良部大橋～クリスタルビーチ】
    【開催時間: 9時30分、13時30分 1日2回】

    3. Sunset SUP

    サンセットビーチは名前のとおり宮古島で美しい夕日が見れるスポットのひとつです。夕日に向かってサップを進め、最高の夕日を海の上から眺めませんか。振り返ると夕日の色に染まったヒルトンホテルを見ることができますよ。
    【参加費: 5,500円】
    【所要時間: 1時間30分】
    【開催場所: サンセットビーチ前】
    【開催時間: 18時30分 (7月、8月、9月)】

    Massage

    みんなで守る「みやこの海」


    「みやこの海」がいつまでも美しくあり続けるよう、私たち一人一人にできることがあります。

    ゴミはきちんと持ち帰ろう

    私たちが出すゴミが自然を壊すことにならないように持ち帰り、決められた場所で分別して捨てましょう。

    美ら海を守りましょう

    サンゴの上に立ったり、サンゴをフィンで折ったりしないよう十分に気をつけましょう。

    限りある資源を大切に使おう

    マイバッグを持つ、物を丁寧に長く使う、いらなくなった物を譲り合う、リサイクル製品を選ぶなど、ゴミを減らして資源を大切に使いましょう。

    エネルギーを大切に使おう

    エアコンの設定を適温にする、こまめに電気を消す、エネルギー消費の少ない製品を選ぶなど、身のまわりのできることから、温暖化防止対策を進めましょう。

    海で遊ぶ前に確認しておきたいこと

    安全に海遊びするためには、事前の情報収集が大切です。海水浴場で泳ぐ場合は、事前に自治体のホームページ等で 公開されている海水浴場の最新情報を確認しておきましょう。


    風

    風向きや風の強さ ( 風速 ) 突風が吹いてくる 危険がないかなどを確認します。

    波

    波の高さ ( 沿岸波浪 ) や干潮 ( 潮の満ち干、 満潮や干潮の時間 ) などを確認します。

    天気・気温

    天気、気温、降水確率などを確認します。台風が 近づいている場合は、台風の位置情報などもチェック。


    海の事故から命を守る三つのポイント


    マリンレジャーを楽しむ際は、ライフジャケットを常時着用する

    海中に転落した際は、ライフジャケットを着用しているかが生死を分ける要素になることがあります。


    防水パック入り携帯電話など連絡手段を確保しておく

    耐水タイプまたは防水パックに入れた携帯電話を 携行しましょう。また場所を正確に知らせられる ように、スマートフォンの位置情報は ON に。

    118 番・NET118 を活用する

    海の事件・事故の緊急通報用電話「118 番」や、 聴覚や発話に障がいがある方向けの緊急通報 サービス「NET118」で海上保安庁に連絡しましょう。


    もしもの時は

    海で遊ぶときは、安全第一で行動することが大切です。天候が急変したり、波が高くなったり、うねりが出てきたときは、すぐに中断して安全な場所に移動しましょう。

    おぼれた人を見かけたときの対処法

    おぼれている人を見かけたら、近くにいる監視員やライフセーバー、まわりの人などに助けを求めてください。また、緊急連絡先に電話をかけて、救助を要請しましょう。
